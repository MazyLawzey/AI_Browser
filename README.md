# AI_Browser

Браузер с ИИ-помощником. Выделяй текст — получай ответ от локальной нейросети.  
Встроенный AI-агент умеет самостоятельно заходить в Moodle, находить тесты и отвечать на вопросы.

## Возможности

- **ИИ-помощник** — выделите текст на любой странице, рядом появится окно с ответом от Ollama
- **Встроенный браузер** — вкладки, навигация колёсиком/тачпадом
- **AI-агент для Moodle** — автоматический вход, поиск тестов, решение вопросов и отправка ответов
- **Наблюдение в реальном времени** — отдельное окно, в котором видно как ИИ выполняет тесты шаг за шагом
- **Ручная помощь** — можно в любой момент остановить агента, помочь с ответом и продолжить
- **Локальный ИИ** — всё работает через Ollama, ничего не уходит в интернет (если не использовать облачную модель)

## Установка

```bash
npm i
```

## Настройка

Скопируйте и отредактируйте `.env`:

```
# Модель Ollama
OLLAMA_MODEL=gemma4:31b-cloud

# Moodle
MOODLE_URL=https://
MOODLE_USERNAME=ваш_логин
MOODLE_PASSWORD=ваш_пароль
AGENT_MODEL=gemma4:31b-cloud
```

Убедитесь, что **Ollama запущена** и нужная модель скачана:

```bash
ollama pull gemma4:31b-cloud
```

## Запуск

### Основное приложение (браузер + ИИ-помощник)

```bash
npm run start
```

### AI-агент для Moodle (отдельное окно)

```bash
npm run agent
```

Откроется окно с Moodle и плавающей панелью управления.  
Нажмите **«Запустить»** — ИИ сам войдёт, найдёт тест и начнёт отвечать.  
Вы можете наблюдать за процессом и в любой момент помочь вручную.

## Как работает агент

1. Открывает страницу входа в Moodle и авторизуется
2. Переходит в дашборд и ищет ссылки на тесты
3. Заходит в тест, читает вопросы со страницы
4. Для каждого вопроса отправляет промпт в Ollama
5. Заполняет ответы (выбор варианта, true/false, текстовое поле)
6. Отправляет тест на проверку

# AI_Browser
Браузер с ии. Буду честен, я создал его чтобы ии отвечал мне на тесты в moodle. вы можете использовать в других целях.

для начала вам надо:
1. npm i
2. открыть .env и редактировать его.
3. запустить ollama
4. npm run start

и все.
